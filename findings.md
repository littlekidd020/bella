# Project Findings: The Boutique Collection

## Research & Discoveries
- [2026-03-21] **UI Patterns**: Chinese massage sites use hyper-specific stats (Height, Weight, Cup) and emoji-rich copy. High-end sites favor dark/gold themes.
- [2026-03-21] **UX Strategy**: Direct messaging (WhatsApp/Telegram/WeChat) is the primary conversion goal. The website is a visual funnel.
- [2026-03-21] **Verification**: "Verified" badges and video loops are critical for trust in this niche.

## Technical Choices
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React / Custom Gold SVGs
